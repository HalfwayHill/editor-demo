# Editor Demo

[项目预览地址](https://halfwayhill.github.io/editor-demo/#/)

## 技术栈

- Vue3
- Vite
- Pinia
- Typescript
- animate.css 动画库
- element-plus 样式库
- echarts 图表库
- lodash 函数库
- mitt 事件收发库
- vue-router 路由库
- ESLint
- Prettier
- husky
- lint-staged

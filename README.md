# Editor Demo

## 技术栈 

- Vue3
- Vite
- Pinia
- Typescript
- animate.css 动画库
- element-plus 样式库
- echarts 图表库
- lodash 函数库
- mitt 事件收发库
- vue-router 路由库

## 项目功能

- 编辑器 - 画布
- 自定义组件
- 拖拽
- 删除组件、调整图层层级
- 放大缩小
- 撤销重做
- 吸附
- 编辑器 - 基础属性栏
- 预览、保存数据
- 编辑器 - 绑定事件
- 编辑器 - 绑定动画
